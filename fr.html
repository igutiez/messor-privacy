Politique de Confidentialité de Messor

Dernière mise à jour : 29 janvier 2026

1. Responsable du traitement

Iñigo Gútiez
Email : igutiez@gmail.com

2. Principe fondamental : VOS DONNÉES VOUS APPARTIENNENT

Messor N'ENVOIE PAS vos données à un serveur.
Toutes les informations sont stockées uniquement sur votre appareil.
Si vous activez la sauvegarde Google Drive (optionnelle), les données vont dans VOTRE compte Google Drive, pas chez nous.
Nous n'avons pas accès à vos données. Nous ne les stockons pas. Nous ne les vendons pas.

3. Informations stockées par l'application (sur votre appareil)

Messor stocke localement sur votre appareil les informations suivantes :

  • Données des animaux : nom, espèce, race, date de naissance, photo, numéro de puce
  • Dossiers médicaux : vaccins, médicaments, visites vétérinaires, chirurgies, documents
  • Journal de santé : poids, température, symptômes, comportement
  • Dépenses : montants, catégories, reçus
  • Rappels : rendez-vous, médicaments programmés
  • Contacts d'urgence : vétérinaires et numéros d'urgence

4. Stockage : 100% local

Toutes les données sont stockées uniquement sur votre appareil. Messor est une application "offline-first" : elle fonctionne entièrement sans connexion internet.

Nous n'avons pas de serveurs. Nous ne stockons aucune de vos données. Nous n'avons aucun moyen d'accéder à vos informations.

5. Google Drive : OPTIONNEL et dans VOTRE compte

Cette fonction est entièrement optionnelle. Si vous choisissez de l'activer :

  • Les données sont sauvegardées dans VOTRE propre compte Google Drive, pas sur les serveurs de Messor
  • Nous utilisons Google Sign-In uniquement pour que VOUS accédiez à VOTRE Google Drive
  • Nous n'accédons qu'à un dossier Messor spécifique dans votre Drive
  • Nous n'avons pas accès à ces données - elles sont dans votre compte Google
  • Vous pouvez désactiver cette fonction ou supprimer le dossier à tout moment

Si vous n'activez pas Google Drive, l'application fonctionne parfaitement avec le stockage local uniquement.

6. Autorisations de l'application

Messor demande les autorisations suivantes, toutes optionnelles selon les fonctions utilisées :

  • Appareil photo : Pour prendre des photos d'animaux et de reçus
  • Stockage : Pour sauvegarder et accéder aux photos et documents
  • Calendrier : Pour ajouter des rappels à votre calendrier
  • Contacts : Pour importer les contacts vétérinaires (lecture seule)
  • Notifications : Pour les rappels de médicaments et rendez-vous
  • Biométrie : Pour protéger l'app avec empreinte digitale ou Face ID
  • Internet : Uniquement pour la synchronisation optionnelle Google Drive

7. Données que nous NE collectons PAS

  • Nous n'utilisons pas d'outils d'analyse ou de suivi
  • Nous ne collectons pas de données de localisation
  • Nous ne partageons pas de données avec des tiers
  • Nous n'affichons pas de publicité
  • Nous ne vendons aucune donnée

8. Achats intégrés

Les dons volontaires sont traités via Google Play ou App Store. Nous n'avons pas accès à vos données de paiement. Ces transactions sont soumises aux politiques de confidentialité de Google et Apple respectivement.

9. Sécurité

Les données stockées sur votre appareil sont protégées par :

  • Le chiffrement du système d'exploitation de l'appareil
  • Le verrouillage biométrique optionnel (empreinte/Face ID)
  • Les sauvegardes Google Drive utilisent la sécurité de votre compte Google

10. Vos droits

Comme toutes les données sont sur votre appareil, vous avez un contrôle total :

  • Accès : Toutes les données sont visibles dans l'application
  • Modification : Vous pouvez modifier n'importe quelle donnée
  • Suppression : Vous pouvez supprimer des données individuelles ou désinstaller l'app
  • Export : Vous pouvez exporter vos données au format Excel ou PDF

11. Enfants

Messor n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations auprès d'enfants.

12. Modifications de cette politique

Nous pouvons mettre à jour cette politique occasionnellement. Nous vous informerons des changements importants via l'application ou par d'autres moyens.
